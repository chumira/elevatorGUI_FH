Diese Applikation simuliert ein Aufzugmodell, welches Befehle über eine serielle Schnittstelle senden und empfangen kann.
Sie nutzt JavaFX.
Das Projekt wurde mit Maven fuer die Abhanegigkeiten entwickelt (siehe pom.xml).
Die .puml Dateien sind PlantUML und dienen der Uebersicht.
Als Javaversion wurde java 21 genutzt.
Zum kompilieren (Intellij): Project Structure -> Project Settings -> Artifacts -> + JAR -> From modules with dependency -> Main Class : Launcher -> extract to the target JAR;
    Build -> build Artifacts -> build
